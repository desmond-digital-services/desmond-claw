## About This Fork

This is the **Desmond Digital Services** fork of OpenClaw, customized for:

- **Business deployments** - Multi-tenant support, industry-specific configurations
- **Local/Cloud/Hybrid hosting** - Deploy anywhere, any infrastructure
- **White-label ready** - Rebrand for your clients
- **Industry templates** - Pre-built configurations for different verticals

### Business Features

| Feature | Description |
|---------|-------------|
| Multi-industry templates | Restaurant, HVAC, Legal, Real Estate, Healthcare, etc. |
| Custom branding | White-label with your company branding |
| Hosting options | Local, cloud, VPS, Docker, Kubernetes |
| Client management | Multi-client deployment support |
| Professional services | Integration, customization, training |

## Getting Started

```bash
# Install
npm install -g desmond-claw@latest
# or: pnpm add -g desmond-claw@latest

# Onboard
desmond-claw onboard --install-daemon
```
